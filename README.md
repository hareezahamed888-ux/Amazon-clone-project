# Amazon Clone

A clean and responsive Amazon-style storefront built with HTML, CSS, and lightweight JavaScript.

## Overview

This project replicates the look and feel of an online shopping experience using a modern product grid and detail pages. It includes:

- A homepage with featured product cards and prices
- Product detail pages with full product information
- Clean responsive styling for desktop and mobile
- Improved shopping app layout and navigation

## Files

- `index.html` - Main storefront page with product listings
- `product.html` - Individual product detail page template
- `style.css` - Shared styles for homepage and detail page
- `amazon_logo.png`, `hero_image.jpg`, `box1_image.jpg`, ... - Product and branding images

## Usage

1. Open `index.html` in a web browser.
2. Click any product card to view the detail page.
3. Use the search bar and category chips to simulate browsing.

## Improvements

This version adds product prices, product detail pages, richer UI styling, and a more realistic shopping app layout.

## Notes

The project is static and works without a backend, making it easy to open locally and customize for demos or learning purposes.
